機能1：Todoを追加する

基本機能を実装する

RED

テストを作成

まずはテストファイルを作成します。 

php artisan make:test TodoListTest --unit

tests/Unit/TodoListTest.phpを開いて、最初のテストを書きます。 

<?php

namespace Tests\Unit;

use PHPUnit\Framework\TestCase;
use PHPUnit\Framework\Attributes\Test;
use App\Services\TodoList;

class TodoListTest extends TestCase
{
    /**
     * Todoを追加できることをテストする
     */
    #[Test]
    public function it_can_add_a_todo()
    {
        // Arrange
        $todoList = new TodoList();

        // Act
        $todoList->add('買い物に行く');

        // Assert
        $todos = $todoList->getAll();
        $this->assertCount(1, $todos);
        $this->assertEquals('買い物に行く', $todos[0]['title']);
    }
}

📖 使用しているアサーション:

assertCount(1, $todos) - 配列の要素数が1であることを確認

assertEquals(...) - Todoのタイトルが正しいことを確認

テストを実行すると失敗します。

php artisan test tests/Unit/TodoListTest.php
GREEN

実装を追加

src/app/Services/TodoList.phpを作成します。 

<?php

namespace App\Services;

class TodoList
{
    private array $todos = [];

    /**
     * Todoを追加する
     */
    public function add(string $title): void
    {
        $this->todos[] = [
            'title' => $title,
            'completed' => false,
        ];
    }

    /**
     * 全てのTodoを取得する
     */
    public function getAll(): array
    {
        return $this->todos;
    }
}

テストを実行すると成功します。

php artisan test tests/Unit/TodoListTest.php

機能2：Todoを完了にする

状態を変更する機能を追加

RED

テストを追加

Todoを完了にする機能のテストを追加します。 

/**
 * Todoを完了できることをテストする
 */
#[Test]
public function it_can_complete_a_todo()
{
    // Arrange
    $todoList = new TodoList();
    $todoList->add('買い物に行く');

    // Act
    $todoList->complete(0); // インデックス0のTodoを完了にする

    // Assert
    $todos = $todoList->getAll();
    $this->assertTrue($todos[0]['completed']);
}
GREEN

実装を追加

TodoList.phpにcompleteメソッドを追加します。 

/**
 * Todoを完了にする
 */
public function complete(int $index): void
{
    if (isset($this->todos[$index])) {
        $this->todos[$index]['completed'] = true;
    }
}

機能3：未完了のTodoを取得

フィルタリング機能を追加

RED

テストを追加

/**
 * 未完了のTodoだけを取得できることをテストする
 */
#[Test]
public function it_can_get_pending_todos_only()
{
    // Arrange
    $todoList = new TodoList();
    $todoList->add('買い物に行く');
    $todoList->add('掃除をする');
    $todoList->add('勉強する');
    $todoList->complete(1); // 2つ目を完了にする

    // Act
    $pending = $todoList->getPending();

    // Assert
    $this->assertCount(2, $pending); // 未完了は2つ
    $this->assertEquals('買い物に行く', $pending[0]['title']);
    $this->assertEquals('勉強する', $pending[1]['title']);
}
GREEN

実装を追加

/**
 * 未完了のTodoだけを取得する
 */
public function getPending(): array
{
    return array_values(array_filter(
        $this->todos,
        fn($todo) => !$todo['completed']
    ));
}

📖 コードの解説:

array_filterで完了していないTodoだけをフィルタリング。
array_valuesでインデックスを振り直しています。

機能4：Todoを削除する

削除機能を追加

RED

テストを追加

/**
 * Todoを削除できることをテストする
 */
#[Test]
public function it_can_remove_a_todo()
{
    // Arrange
    $todoList = new TodoList();
    $todoList->add('買い物に行く');
    $todoList->add('掃除をする');

    // Act
    $todoList->remove(0);

    // Assert
    $todos = $todoList->getAll();
    $this->assertCount(1, $todos);
    $this->assertEquals('掃除をする', $todos[0]['title']);
}
GREEN

実装を追加

/**
 * Todoを削除する
 */
public function remove(int $index): void
{
    if (isset($this->todos[$index])) {
        unset($this->todos[$index]);
        $this->todos = array_values($this->todos); // インデックスを振り直す
    }
}

php artisan test tests/Unit/TodoListTest.php

🚀 さらに機能を追加してみよう

課題3：全Todoをクリア

ヒント：配列を空にします。テストではassertEmpty()が使えます。

search($keyword)メソッドを実装してください。

ヒント：str_contains()関数で部分一致検索ができます。

💡 TDDで実装する手順

  1. まずテストを書く（RED）
  2. テストを実行して失敗を確認
  3. 最小限の実装でテストを通す（GREEN）
  4. 必要に応じてRefactor
  5. 次の機能へ
