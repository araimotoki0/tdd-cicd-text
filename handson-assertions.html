検証編：様々なアサーション

引き算・掛け算・割り算を追加しながらアサーションメソッドを学ぶ

アサーションとは？

📘 定義

アサーション（Assertion）とは、
「〜であることを確認する」という検証のことです。

例えば、assertEquals(5, $result)は
「$resultが5と等しいことを確認する」という意味です。

PHPUnitには様々なアサーションメソッドがあり、
状況に応じて適切なものを使い分けます。

💡 このセクションの進め方

Calculatorクラスに引き算・掛け算・割り算を追加しながら、
それぞれに適したアサーションメソッドを1つずつ学んでいきます。

引き算：assertTrue / assertFalse

真偽値の検証を学ぶ

1

RED：テストを追加

CalculatorTest.phpに引き算のテストを追加します。 

/**
 * 引き算ができることをテストする
 */
#[Test]
public function it_can_subtract_two_numbers()
{
    // Arrange
    $calculator = new Calculator();

    // Act
    $result = $calculator->subtract(10, 3);

    // Assert - 結果が正の数であることを確認
    $this->assertTrue($result > 0);

    // Assert - 結果が負の数でないことを確認
    $this->assertFalse($result < 0);

    // Assert - 正確な値も確認
    $this->assertEquals(7, $result);
}

📖 新しいアサーション:

assertTrue($condition)

条件がtrueであることを確認します。

assertFalse($condition)

条件がfalseであることを確認します。

テストを実行すると失敗します（subtractメソッドがまだ存在しない）。

php artisan test tests/Unit/CalculatorTest.php
2

GREEN：実装を追加

Calculator.phpにsubtractメソッドを追加します。 

/**
 * 2つの数値を減算する
 *
 * @param int|float $a 被減数
 * @param int|float $b 減数
 * @return int|float 減算結果
 */
public function subtract(int|float $a, int|float $b): int|float
{
    return $a - $b;
}

テストを実行すると成功します。

php artisan test tests/Unit/CalculatorTest.php

掛け算：assertGreaterThan / assertLessThan

大小比較の検証を学ぶ

1

RED：テストを追加

CalculatorTest.phpに掛け算のテストを追加します。 

/**
 * 掛け算ができることをテストする
 */
#[Test]
public function it_can_multiply_two_numbers()
{
    // Arrange
    $calculator = new Calculator();

    // Act
    $result = $calculator->multiply(3, 4);

    // Assert - 結果が10より大きいことを確認
    $this->assertGreaterThan(10, $result);

    // Assert - 結果が15より小さいことを確認
    $this->assertLessThan(15, $result);

    // Assert - 正確な値も確認
    $this->assertEquals(12, $result);
}

📖 新しいアサーション:

assertGreaterThan($expected, $actual)

$actualが$expectedより大きいことを確認します。

assertLessThan($expected, $actual)

$actualが$expectedより小さいことを確認します。

2

GREEN：実装を追加

Calculator.phpにmultiplyメソッドを追加します。 

/**
 * 2つの数値を乗算する
 *
 * @param int|float $a 第1の乗数
 * @param int|float $b 第2の乗数
 * @return int|float 乗算結果
 */
public function multiply(int|float $a, int|float $b): int|float
{
    return $a * $b;
}

割り算：例外のテスト

エラーケースを検証する

1

RED：正常系のテスト

まずは通常の割り算のテストを追加します。 

/**
 * 割り算ができることをテストする
 */
#[Test]
public function it_can_divide_two_numbers()
{
    // Arrange
    $calculator = new Calculator();

    // Act
    $result = $calculator->divide(10, 2);

    // Assert
    $this->assertEquals(5, $result);
}
2

RED：例外のテスト

0で割ろうとした場合、例外が投げられることをテストします。 

/**
 * 0で割ると例外が投げられることをテストする
 */
#[Test]
public function it_throws_exception_when_dividing_by_zero()
{
    // Arrange
    $calculator = new Calculator();

    // Assert - 例外が投げられることを期待
    $this->expectException(InvalidArgumentException::class);
    $this->expectExceptionMessage('0で割ることはできません');

    // Act - 例外が投げられるはず
    $calculator->divide(10, 0);
}

📖 例外テスト用のメソッド:

expectException($exception)

指定した例外が投げられることを期待します。

expectExceptionMessage($message)

例外メッセージが指定した文字列を含むことを期待します。

💡 重要: 例外テストでは、expectExceptionをActの前に書きます。

3

GREEN：実装を追加

Calculator.phpにdivideメソッドを追加します。 

/**
 * 2つの数値を除算する
 *
 * @param int|float $a 被除数
 * @param int|float $b 除数
 * @return int|float 除算結果
 * @throws \InvalidArgumentException 除数が0の場合
 */
public function divide(int|float $a, int|float $b): int|float
{
    if ($b === 0) {
        throw new \InvalidArgumentException('0で割ることはできません');
    }

    return $a / $b;
}

全てのテストを実行します。

php artisan test tests/Unit/CalculatorTest.php

その他の便利なアサーション

Null チェック

assertNull($value)

assertNotNull($value)

型チェック

assertIsInt($value)

assertIsFloat($value)

assertIsString($value)

assertIsArray($value)

配列関連

assertCount($n, $array)

assertContains($item, $array)

assertEmpty($array)

assertNotEmpty($array)

文字列関連

assertStringContains($needle, $haystack)

assertStringStartsWith($prefix, $string)

assertStringEndsWith($suffix, $string)

💡 アサーション選びのコツ

「何を確認したいか」を明確にして、それに最も適したアサーションを選びましょう。
例えば、「5と等しい」ならassertEquals、「5より大きい」ならassertGreaterThanを使います。

検証編のまとめ

学んだアサーション

  • • 異常系のテスト（エラーケース）
  • • 境界値のテスト（0、負の数など）
